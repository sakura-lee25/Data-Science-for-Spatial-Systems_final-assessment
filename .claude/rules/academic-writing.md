# 学术写作规范

## 语言
- 使用正式学术英语
- 避免第一人称，使用被动语态: "The data was analysed" not "I analysed"
- 使用英式拼写 (analyse, behaviour, colour, centre)

## 引用格式
- 使用 Harvard/APA 引用格式
- 文内引用: (Author, Year) 或 Author (Year)
- 所有数据来源必须引用

## 图表
- 所有图表必须有编号和标题 (Figure 1: ...)
- 表格标题在上方，图片标题在下方
- 注明数据来源
- 300 DPI 以上

## 统计报告
- 报告格式: "A significant effect was found (χ² = 12.34, df = 3, p < .001)"
- 保留小数位: p 值 3 位，其他统计量 2 位
- 效应量必须报告
